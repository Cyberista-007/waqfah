# Musnad Ahmad API (Ar-Risala Edition) 📚

هذا المستودع يحتوي على **مسند الإمام أحمد بن حنبل** كاملاً (أكثر من 26,000 حديث)، موزعاً ومصنفاً وفقاً لطبعة **مؤسسة الرسالة** (تحقيق شعيب الأرنؤوط) المكونة من 52 مجلداً.

## ✨ المميزات:
- **تصنيف دقيق:** مقسم إلى 52 ملف JSON تطابق المجلدات الورقية.
- **عناوين علمية:** كل مجلد يحتوي على أسماء الصحابة (المسانيد) المندرجة تحته.
- **نصوص مشكولة:** الأحاديث متوفرة بالتشكيل الكامل لسهولة القراءة والبحث.
- **خفيف وسريع:** مصمم ليكون API ثابت (Static API) سهل الاستخدام في تطبيقات الويب والجوال.

## 🚀 كيفية الاستخدام (Usage):
يمكنك الوصول للبيانات مباشرة عبر الروابط التالية:
- **الفهرس (Index):** `https://raw.githubusercontent.com/[YOUR_USERNAME]/Musnad-Ahmad-API/main/sections.json`
- **المجلدات (Volumes):** `https://raw.githubusercontent.com/[YOUR_USERNAME]/Musnad-Ahmad-API/main/sections/{id}.json`
