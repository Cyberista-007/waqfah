# منصة وقفة العلمية (Waqfa Platform)

"وقفة" هي منصة ويب تعليمية إسلامية شاملة، مصممة لتكون مكتبة مركزية للمحتوى الإسلامي الموثوق. تهدف المنصة إلى تيسير الوصول إلى محاضرات ودروس وسلاسل علمية لنخبة من العلماء والمشايخ، مع توفير أدوات تفاعلية تساعد المستخدم على التعلم والنمو.

## ✨ الميزات الرئيسية

### للمستخدمين:

*   **مكتبة شاملة:** تصفح واستمع إلى آلاف المحاضرات والسلاسل والكتب والبرامج (القنوات).
*   **بحث متقدم:** ابحث بسهولة في جميع أنواع المحتوى، مع إمكانية الفلترة حسب اللغة والمدة والبحث داخل التفريغ النصي للمحاضرات.
*   **تجربة استماع متقدمة:** مشغل صوتي عائم ومستمر، ومشغل فيديو احترافي (Plyr.io)، مع دعم للحفظ التلقائي لموضع الاستماع.
*   **مكتبتي (الملف الشخصي):**
    *   **المفضلة:** احتفظ بالمحاضرات التي أعجبتك.
    *   **قوائم التشغيل:** أنشئ قوائم تشغيل خاصة أو شاركها مع الآخرين.
    *   **متابعة البرامج:** تابع برامجك المفضلة ليصلك كل جديد.
    *   **سجل الاستماع:** أكمل الاستماع من حيث توقفت.
*   **محاسبة النفس:** أداة تفاعلية فريدة لتتبع العبادات اليومية (الصلوات، الأذكار، السنن) واكتساب النقاط.
*   **التحديات ولوحة الصدارة:** شارك في تحديات الاستماع، واجمع النقاط والأوسمة، وتنافس في الخيرات.
*   **دعم المشروع:** صفحة مخصصة لدعم استمرارية المشروع من خلال التبرعات، مع جدار للداعمين ومستويات دعم تقديرية.
*   **تخصيص المظهر:** اختر من بين العديد من الثيمات والخطوط لتخصيص تجربتك البصرية.

### للمديرين:

*   **لوحة تحكم شاملة:** نظرة عامة على إحصائيات الموقع، وروابط سريعة لإدارة جميع الأقسام.
*   **إدارة المحتوى:** إضافة وتعديل وحذف البرامج، السلاسل، المحاضرات، والكتب بسهولة.
*   **استيراد من يوتيوب:** أداة قوية لاستيراد قناة أو قائمة تشغيل كاملة من يوتيوب، مع جلب البيانات الوصفية تلقائيًا.
*   **إدارة الميزات:** التحكم الكامل في العناصر المثبتة بالصفحة الرئيسية، التحديات، الأوسمة، جدول الدروس، والأسئلة الشائعة.
*   **إدارة الموقع:** التحكم في إعدادات المظهر، شريط الإعلانات، حملات التبرع، وإدارة المستخدمين.

## 🚀 التقنيات المستخدمة

*   **إطار العمل:** [Next.js](https://nextjs.org/) (مع App Router)
*   **الخلفية وقاعدة البيانات:** [Firebase](https://firebase.google.com/) (Firestore, Authentication, Storage)
*   **التصميم وواجهة المستخدم:** [Tailwind CSS](https://tailwindcss.com/) و [ShadCN UI](https://ui.shadcn.com/)
*   **مشغل الوسائط:** [Plyr.io](https://plyr.io/)
*   **الذكاء الاصطناعي:** [Genkit](https://firebase.google.com/docs/genkit) (لتوصيات المحتوى)

## 🛠️ التشغيل محليًا

1.  **استنساخ المستودع:**
    ```bash
    git clone [رابط_المستودع]
    cd [اسم_المجلد]
    ```

2.  **تثبيت التبعيات:**
    ```bash
    npm install
    ```

3.  **إعداد متغيرات البيئة:**
    *   أنشئ ملفًا باسم `.env.local` في جذر المشروع.
    *   للاتصال بقاعدة بيانات Firestore محليًا (لعرض البيانات)، اتبع التعليمات في `DEPLOYMENT.md` للحصول على مفتاح حساب الخدمة (Service Account) وأضفه إلى ملف `.env.local` كالتالي:
        ```
        FIREBASE_SERVICE_ACCOUNT='{...محتوى_ملف_الـJSON_هنا...}'
        ```
    *   لتمكين ميزات استيراد يوتيوب، اتبع التعليمات في `docs/youtube-api-key.md` وأضف مفتاحك:
        ```
        YOUTUBE_API_KEY="AIza..."
        ```

4.  **تشغيل خادم التطوير:**
    ```bash
    npm run dev
    ```

5.  افتح [http://localhost:3000](http://localhost:3000) في متصفحك.
