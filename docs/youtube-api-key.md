# كيفية الحصول على مفتاح YouTube Data API v3

لتمكين الميزات التي تتطلب جلب بيانات من يوتيوب (مثل استيراد بانر القناة أو تحديث بيانات المحاضرات)، ستحتاج إلى إنشاء مفتاح لواجهة برمجة التطبيقات (API Key) من Google Cloud.

اتبع الخطوات التالية بعناية:

---

## الخطوة 1: الذهاب إلى Google Cloud Console

1.  افتح متصفحك واذهب إلى [**Google Cloud Console**](https://console.cloud.google.com/).
2.  قم بتسجيل الدخول باستخدام حساب Google الخاص بك.

---

## الخطوة 2: إنشاء مشروع جديد (أو اختيار مشروع حالي)

1.  في أعلى الصفحة، بجوار شعار Google Cloud، سترى قائمة منسدلة للمشاريع. انقر عليها.
2.  في النافذة التي تظهر، انقر على **"New Project"** (مشروع جديد).
3.  أدخل اسمًا لمشروعك (مثلاً: `My-Youtube-API-Project`) ثم انقر على **"Create"** (إنشاء).
4.  انتظر بضع لحظات حتى يتم إنشاء المشروع، ثم تأكد من أنك قمت باختياره من القائمة المنسدلة للمشاريع.

---

## الخطوة 3: تفعيل YouTube Data API v3

1.  من قائمة التنقل الجانبية (انقر على أيقونة الثلاثة خطوط ☰ إذا كانت مخفية)، اذهب إلى **"APIs & Services"** (واجهات برمجة التطبيقات والخدمات) > **"Library"** (المكتبة).
2.  في شريط البحث، اكتب `YouTube Data API v3` واضغط Enter.
3.  انقر على النتيجة التي تظهر.
4.  في الصفحة التالية، انقر على زر **"Enable"** (تفعيل).

---

## الخطوة 4: إنشاء مفتاح API

1.  بعد تفعيل الواجهة، سيتم توجيهك إلى صفحة إدارتها. من القائمة الجانبية، انقر على **"Credentials"** (بيانات الاعتماد).
2.  في أعلى الصفحة، انقر على **"+ CREATE CREDENTIALS"** (إنشاء بيانات اعتماد) واختر **"API key"** (مفتاح API).
3.  سيتم إنشاء مفتاحك على الفور وسيظهر في نافذة منبثقة. انقر على أيقونة النسخ لنسخ المفتاح.

---

## الخطوة 5: (مهم جدًا) تقييد المفتاح

لحماية حسابك ومنع الاستخدام غير المصرح به، يجب عليك تقييد المفتاح.

1.  في النافذة المنبثقة التي ظهر فيها مفتاحك، انقر على **"EDIT API KEY"** (تعديل مفتاح API). (إذا أغلقتها، يمكنك النقر على اسم المفتاح في قائمة بيانات الاعتماد).
2.  تحت قسم **"API restrictions"** (قيود واجهة برمجة التطبيقات):
    *   اختر **"Restrict key"** (تقييد المفتاح).
    *   في القائمة المنسدلة، ابحث عن `YouTube Data API v3` وقم باختياره.
    *   انقر على **"OK"**.
3.  انقر على زر **"Save"** (حفظ) في الأسفل.

---

## الخطوة 6: إضافة المفتاح إلى مشروعك

1.  افتح ملف `.env` في جذر مشروعك.
2.  ستجد سطراً جديداً قمت بإضافته لك:
    ```
    # YOUTUBE_API_KEY="ضع_المفتاح_هنا"
    ```
3.  احذف علامة `#` من بداية السطر، ثم الصق المفتاح الذي نسخته مكان `ضع_المفتاح_هنا`. يجب أن يبدو السطر هكذا:
    ```
    YOUTUBE_API_KEY="AIzaSy...Your...Key...Here"
    ```
4.  احفظ الملف. ستحتاج إلى إعادة تشغيل خادم التطوير المحلي لكي يتم تطبيق التغييرات.

الآن، يجب أن تعمل الميزات التي تعتمد على واجهة يوتيوب بشكل صحيح.
