# **App Name**: Amjad Samir Lectures

## Core Features:

- Lecture Browsing: Allow users to browse lectures by series, topic, or recency.
- Content Filtering & Sorting: Filter lectures and series by category, and sort by date, popularity, or title.
- Audio Playback: Enable in-app audio playback with a floating player for continuous listening.
- Interactive Transcripts: Provide interactive, clickable transcripts that sync with the audio playback.
- User Authentication: Allow users to sign in anonymously using Firebase to enable features like saving favorites and rating lectures.
- Admin Dashboard: Provide a password-protected admin interface for managing lectures, series, and users.
- Personalized Recommendations: The application employs an AI tool that recommends lectures of interest to a user, based on their viewing history.

## Style Guidelines:

- Primary color: Soft lavender (#D0BFFF) for a calming, intellectual feel, fitting for educational content.
- Background color: Very light gray (#F5F5F5), providing a neutral backdrop to ensure readability and focus.
- Accent color: Pale violet (#E6D6FF) subtly complements the primary hue, to create visual interest without overwhelming.
- Body and headline font: 'Alegreya', a serif font lending an elegant, intellectual, contemporary feel.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use clear, modern icons for navigation and actions. Icons should be monochromatic and aligned with the color scheme.
- Employ a clean, spacious layout with generous padding and margins to improve readability and focus. Content should be well-organized and easy to navigate.
